Workstation

Workstation is a pesronal project based on the understanding of versioning and collaboration tools. This model is made in C# and is really simplified compared to heavy systems like SLACK, GitHub, Team Foundation and so on.
New componants, althought, emerged from themselves and most of them will be written in Go to simplify the process.

Most of the structure is not complete yet, especially security and the main features. The browser version is focused yet Universal Windows Project and Mobile Projects (Android and probably IOS) are planned. The actual version is barely in Alpha stage and only a "template" is planned for UWP - the real implementation has not started yet.

Briefly, it regroups several services. The authentication one using ASP.NET Core Rest and OAuth0-like authentication (using Windows Identity Tokens), the data access is a WCF web service (that will turn into a micro-service pattern) connected to a SQL Server.

For more informations, please see the repository: https://github.com/DaemonToolz/WorkStation
